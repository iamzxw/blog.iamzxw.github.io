一个数组分成，左右两部分，求|max左 - max右|最大值问题。（业务相关）


___
一个数组，子数组的累加和最大:

```java

int[] arr = {3,2,-4,-5,6,3,-1,7}

int cur_sum = 0;
int max = Integer.MIN_VALUE;

for(int i = 0; i<arr.length; i++){
    cur_sum += arr[i];
    // max = cur_sum > max ? cur_sum:max;
    max = Math.max(max, cur_sum);

    cur_sum = cur_sum < 0 ? 0 : cur_sum;
```
